Weibo Crawler

Obtain repost nodes and links with given url

Totally, we use Weibo Crawler to get about 2 million original weibo repost information. As for a single message, the largest repost number is 1208482.

Data Cleaning

Clean json data for a clear view

The raw data obtained from crawler is pretty messy and poor structured. So, we use python to apply a BFS to the whole graph, in this way, we have better knowledge of children of each node and other statistic information.

D3

Visualization platform

D3.js is a JavaScript library for manipulating documents based on data. We use D3 to visualize the Weibo Network by showing different layers of the repost and the basic information of each user. We use various color and different node size to differentiate users.
